# ICS_Employee

Для начала работы необходимо восстановить базу данных из бэкапа (DataBase -> ICSDB.bak). 
Если версионность не позволяет воспользоваться бэкапом, тогда следует выполнить скрипты, лежащие в папке Database, в следующем порядке:
1. Create_DB_Tables.sql
2. InsertTestData.sql
3. StoredProcedure.sql

Далее необходимо настроить конфигурацию соединения с сервером: 
в корневой папке открыть ярлык ConnectionConfig.txt и изменить название локального сервера (Data Source) на соответствующее.

Запустить ярлык ICS_Employee.exe

Читаем небольшое описание в файле Description.docx
